删除的文件记录

为了配合极简主义理念，我要删除

  • 重复的文件
  • 失效过时的文件
  • 网上能轻易下到的文件
  • 可留可不留的文件

这也是经过奔跑的奶酪的博客启发的行动

接下来在此，我会记录下我选择删除的东西和原因，方便以后的回顾和参考。

3D人物模型DesignDoll汉化版

  • 可留可不留、网上极易下到，用途是在绘画时做人体参考
  • 这是一种溢出的功能，真正的绘画大师本身不需要这种诡异东西
  • 大家可以直接自己摆出动作然后拍照或者网上找动作，所以不需要这个软件

CAJ

  • 知网论文阅读器
  • 网上轻易下到
  • 已经毕业，不需要知网论文

CorelDRAW12简体中文版

  • 版本过老
  • 网上轻易下载

GIF录制Honeycam

  • 我不需要
  • 网上轻易下到
  • 本身不好用

prezi大礼包

  • OFFICE就能满足
  • 不主流的幻灯片软件
  • 网上轻易下到

Uninstall Tool v3.5.7 Build 5610

  • 我真的需要吗
  • 不会用，需要学习
  • 有同类软件

VOCALOID3汉化版加中文洛天依语音库附教程整理

  • 过时
  • 需要时再下
  • 没用过

按键精灵9

  • 不需要
  • 脚本软件，主要用途：鼠标连点
